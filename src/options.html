Recoll Search Indexer Options
 Download subdirectory 

 Subdirectory of the Downloads folder where we store temporary page copies (e.g. Automatic-Web-Page-Downloader ). This is empty by default. It can't be a hidden directory (name beginning with a '.' on Linux). If you set a value, you will also need to set the webdownloadsdir recoll configuration variable to the full location path in recoll.conf, e.g.: webdownloadsdir=/home/mylogin/Downloads/Automatic-Web-Page-Downloader.

 URL selection 

There are two sets of rules, for selecting URLs, or for rejecting them, for automatic downloading. These do not apply when you click the icon: the extension will always try to download in this case.

If both sets are empty, pages are downloaded by default (the two checkboxes below have no effect).
